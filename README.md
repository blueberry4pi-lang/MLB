# MLB Analyzer

This project analyzes Major League Baseball data to provide insights and statistics.

## Features
- Data collection from MLB APIs
- Statistical analysis of player and team performance
- Visualization of trends and patterns
- Prediction models for future performance

## Getting Started
1. Install dependencies
2. Configure API keys
3. Run analysis scripts

## Project Structure
- `src/` - Source code
- `data/` - Data files
- `notebooks/` - Jupyter notebooks for analysis
- `tests/` - Test files
